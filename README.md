Kasio Calculator
Kasio Calculator is a sleek, Apple-inspired web calculator built with HTML, CSS, and JavaScript.
It supports everyday arithmetic, an optional scientific mode, sample calculations, and a full history of results.

Features
Apple-style dark calculator UI
Basic operations (add, subtract, multiply, divide, percent)
Toggleable scientific functions (sin, cos, tan, square root, square, pi)
Run Sample button to generate random example calculations
History panel showing all calculations, including samples
Responsive layout with centered calculator and side history panel on desktop
